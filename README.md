Nearty - Titip Beli Jajanan Lewat Driver Terdekat

Aplikasi berbasis Laravel 13 untuk titip beli jajanan lewat sesama pengguna yang lagi jadi driver dadakan — tanpa perlu keluar kost, tanpa ojek online untuk order sekecil itu.

Fitur Utama


Autentikasi & manajemen role-permission (Admin & User via Spatie)
Buat titipan: lokasi warung, daftar barang, alamat antar, metode bayar (QR/COD)
Mode Driver: toggle available, lihat & ambil titipan terdekat, update status
CRUD: Titipan, Item Titipan, Kategori
Dashboard statistik + grafik interaktif
Search, sort, filter, pagination
REST API (dokumentasi endpoint di bawah)
