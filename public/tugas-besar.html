Fitur Wajib

Semua fitur di bawah ini harus ada dalam proyek

SEMUA HARUS ADA
🗃
Migration & Seeder
Struktur tabel harus dibuat via migration, bukan manual. Seeder wajib ada agar demo bisa dijalankan dari database kosong.
🔑
Autentikasi
Login, Register, Logout. Gunakan Laravel Breeze. Tambahkan validasi input dan proteksi CSRF.
🛡
Role & Permission
Minimal 2 role: Admin dan User/Staff. Akses halaman dibatasi sesuai role menggunakan middleware Spatie.
📋
3 Halaman CRUD
Minimal tiga entitas dengan fitur Create, Read (list), Update, Delete lengkap beserta konfirmasi hapus.
📊
Dashboard
Tampilkan statistik ringkas (jumlah data), grafik tren, ringkasan terbaru, dll. Query harus efisien.
🔍
Search & Pagination
Tabel data harus dapat dicari (search) dan dipaginasi. Tidak boleh menampilkan semua data sekaligus.
📱
UI/UX Responsif
Tampilan harus nyaman di desktop dan mobile. Konsistensi komponen penting.
🔌
REST API
Minimal 2 resource API (JSON) dengan endpoint CRUD. Sertakan dokumentasi endpoint di README.
👥
